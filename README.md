## 概念

大语言模型从 Scaling（规模扩展） 到 Reasoning（推理能力） 再到 Agentic（智能体化） 的三阶段演进：Scaling 通过参数、数据和算力扩张建立知识底座；Reasoning 借助后训练、链式思维和强化学习提升逻辑推导能力；Agentic 则进一步强调模型与工具、环境和反馈系统交互，形成“感知—行动—反馈—调整”的闭环，以完成真实世界中的复杂长期任务。
